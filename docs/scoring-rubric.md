# Scoring Rubric

Total skor: **100 poin** (9 kriteria positif + 1 penalti).

Gunakan rubric ini di Step 2 (Scene Candidates) untuk memberi skor pada
tiap kandidat scene 15–40 detik.

---

## Kriteria

| # | Kriteria | Bobot | Pertanyaan diagnostik |
|---|---|---|---|
| 1 | **Immediate Hook** | 0–20 | Apakah 3 detik pertama scene ini bisa langsung memicu pertanyaan / visual shock? |
| 2 | **Understandable Without Context** | 0–15 | Penonton US tanpa pernah nonton show ini akan paham? |
| 3 | **High Stakes / Mystery** | 0–15 | Ada nyawa, hubungan, kebebasan, atau kebenaran yang dipertaruhkan? |
| 4 | **Visual Clarity Potential** | 0–10 | Apakah visual cukup jelas (wajah, lokasi, aksi) tanpa butuh banyak insert? |
| 5 | **Escalation** | 0–10 | Apakah stakes naik di sepanjang scene, bukan flat? |
| 6 | **Twist / Reveal** | 0–10 | Ada pembalikan asumsi atau reveal yang berbeda dari ekspektasi? |
| 7 | **VO Narration Potential** | 0–10 | Bisa ditulis VO 160–230 kata yang dramatik tanpa mengarang isi kepala karakter? |
| 8 | **Originality of Angle** | 0–5 | Apakah angle ini terasa fresh, atau sudah jenuh di feed? |
| 9 | **Emotional / Human Stakes** | 0–5 | Ada elemen manusia (anak, orang tua, korban, pasien) yang bisa dirasakan secara emosional? |
| P | **Platform Risk Penalty** | -10..0 | Konten sensitif (terror, agama, mass death, graphic injury, race/politics, self-harm)? |

Total = (1+2+3+4+5+6+7+8+9) - |Platform Risk Penalty|.

---

## Cara Menilai (Scoring Guidelines)

### 1. Immediate Hook (0–20)

| Skor | Indikator |
|---|---|
| 18–20 | Visual shock + pertanyaan implisit dalam 1.5 detik. Tidak butuh teks tambahan untuk paham urgensi. |
| 14–17 | Hook visual jelas, tapi butuh 1 baris VO untuk landing. |
| 10–13 | Hook ada tapi lambat (3–5s buildup). |
| 5–9 | Setup-heavy. Penonton kemungkinan swipe sebelum hook landing. |
| 0–4 | Tidak ada hook. Pembukaan flat. |

### 2. Understandable Without Context (0–15)

| Skor | Indikator |
|---|---|
| 13–15 | 100% standalone. Tidak butuh tahu show, season, hubungan karakter. |
| 10–12 | Butuh 1 kalimat setup di VO. |
| 6–9 | Butuh 2–3 kalimat setup. Risiko membosankan. |
| 0–5 | Hanya make-sense kalau follower show. Tolak. |

### 3. High Stakes / Mystery (0–15)

| Skor | Indikator |
|---|---|
| 13–15 | Nyawa / kebebasan / hubungan inti dipertaruhkan. Ada misteri jelas. |
| 9–12 | Stakes jelas tapi sedang. Misteri implisit. |
| 4–8 | Stakes rendah (pertengkaran ringan, dilema kecil). |
| 0–3 | Tidak ada stakes. Cuma slice of life. |

### 4. Visual Clarity Potential (0–10)

| Skor | Indikator |
|---|---|
| 9–10 | Wajah ekspresif, lokasi ikonik, aksi jelas. Cukup KEEP minim insert. |
| 6–8 | Cukup jelas tapi butuh beberapa insert / B-roll. |
| 3–5 | Visual kabur, talking head, butuh banyak edit. |
| 0–2 | Visual gelap, blur, atau tidak ada wajah / aksi. |

### 5. Escalation (0–10)

| Skor | Indikator |
|---|---|
| 9–10 | Stakes naik 2–3x dari awal ke akhir scene. |
| 6–8 | Naik sekali (1 lonjakan). |
| 3–5 | Flat tapi tetap intens. |
| 0–2 | Menurun atau membosankan di tengah. |

### 6. Twist / Reveal (0–10)

| Skor | Indikator |
|---|---|
| 9–10 | Twist landing di scene ini sendiri (tidak butuh konteks luar). |
| 6–8 | Twist ada tapi butuh 1 line VO untuk dijelaskan. |
| 3–5 | Twist ada tapi butuh konteks dari scene sebelumnya. |
| 0–2 | Tidak ada twist / reveal. |

### 7. VO Narration Potential (0–10)

| Skor | Indikator |
|---|---|
| 9–10 | Mudah ditulis VO dramatik 160–230 kata tanpa mengarang isi kepala. |
| 6–8 | Bisa, butuh kerja kreatif untuk pacing. |
| 3–5 | VO terasa dipaksakan atau redundant dengan dialog asli. |
| 0–2 | VO tidak menambah apa-apa. Lebih cocok dialogue clipping (dan kita tidak melakukan itu). |

### 8. Originality of Angle (0–5)

| Skor | Indikator |
|---|---|
| 4–5 | Angle fresh, jarang dilihat di feed. |
| 2–3 | Angle umum tapi masih engaging. |
| 0–1 | Angle jenuh ("lab partner reveal", "twin twist") tanpa diferensiasi. |

### 9. Emotional / Human Stakes (0–5)

| Skor | Indikator |
|---|---|
| 4–5 | Ada anak, orang tua, pasien terminal, atau korban yang ditampilkan secara emosional. |
| 2–3 | Stakes manusia ada tapi tidak diperkuat secara visual. |
| 0–1 | Stakes hanya konsep (uang, kekuasaan tanpa wajah manusia). |

### P. Platform Risk Penalty (-10..0)

| Penalti | Indikator |
|---|---|
| 0 | Aman, tidak menyinggung kategori sensitif. |
| -2..-4 | Sentuh tema sensitif tapi bisa di-repackage netral (mis. injury yang tidak grafis). |
| -5..-7 | Tema sensitif jelas (death, severe injury, family tragedy). Butuh repackaging hati-hati. |
| -8..-10 | Tema sangat sensitif (terror, religion, race, self-harm explicit, mass casualty graphic). Pertimbangkan tolak scene. |

Lihat [`platform-sensitivity.md`](platform-sensitivity.md) untuk daftar
rinci dan repackaging strategy.

---

## Grade

| Grade | Range | Action |
|---|---|---|
| **A** | 85–100 | Strong candidate. Eksekusi penuh. |
| **B** | 70–84 | Usable kalau VO kuat. Hati-hati di execution. |
| **C** | 55–69 | Lemah atau terlalu context-dependent. Pakai hanya kalau tidak ada A/B. |
| **Reject** | <55 | Jangan dipakai. Cari kandidat lain. |

## Contoh Penghitungan

Scene: paramedic membuat keputusan triage di TKP kecelakaan, dengan
reveal bahwa korban kedua adalah orang yang dia kenal.

| Kriteria | Skor |
|---|---|
| Immediate Hook | 17 |
| Understandable Without Context | 12 |
| High Stakes | 14 |
| Visual Clarity | 8 |
| Escalation | 9 |
| Twist | 9 |
| VO Potential | 8 |
| Originality | 3 |
| Emotional Stakes | 5 |
| Platform Risk | -3 (graphic injury risk) |
| **Total** | **82 (Grade B)** |

Verdict: usable. Repackaging penalty ke -1 dengan menghindari close-up
luka berat → skor naik ke 84, masih Grade B tapi lebih aman.

---

## Anti-Cheat: Rubric Discipline

LLM cenderung memberi "everything is 7/10". Hindari:

- Skor 7+ tanpa justifikasi spesifik di kolom `reason`.
- Penalti 0 untuk scene yang jelas sensitif.
- Memberi skor sama untuk 3 kandidat berbeda — itu tanda kurang
  diferensiasi.

Aturan praktis: minimal 1 dari 6 kandidat harus dapat penalti `Platform
Risk` ≠ 0 jika kategorinya melibatkan death/injury/crime.
